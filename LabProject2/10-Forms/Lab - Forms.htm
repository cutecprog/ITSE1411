ITSE 1411 Beginning Web Page Programming

Last revised: 03/12/2011

Note: you will submit these lab files as part of Lab Project 2.

Learning Outcomes1

  • Demonstrate the following (X)HTML programming skills:
    • Create a form with various form elements including:
      • button, checkbox, hidden, image, password, radio, reset, select, submit, text, textarea.
    • Specify form properties.
    • Send data from a form to a CGI script on a web server.
    • Send form data without using a CGI script.
    • Enhance form elements.
  • Demonstrate the correct use and syntax of the following (X)HTML elements:
    • <form>    <input>    <button>    <select>    <option> & <optgroup> within <select> element    <textarea>    <label>    <fieldset>    <legend> within <fieldset> element
  • Demonstrate the correct use and syntax of the following frame attributes:
    • Element: form
      Attributes: action    enctype    method
    • Element: input
      Attributes: type    name    value    size    maxlength    checked    src    alt
    • Element: button
      Attributes: name    value    type
    • Element: select
      Attributes: size    multiple    name    size    tabindex
    • Element: option
      Attributes: selected    value
    • Element: optgroup
      Attribute: label
    • Element: textarea
      Attributes: name    rows    cols
    • Element: label
      Attribute: for
