ITSE 1411 Beginning Web Page Programming

Last revised: 03/12/2011

Note: you will submit these lab files as part of Lab Project 2.

Learning Outcomes1

  • Demonstrate the following (X)HTML programming skills:
    • Create frames for a Web site.
    • Control the appearance and placement of frames.
    • Control the behavior of hyperlinks on a page with frames.
    • Use target names to specify the target for a hypertext link.
    • Create a web page that is viewable both by browsers that support frames and by those that do not support frames.
    • Change the appearance of frames.
  • Demonstrate the correct use and syntax of the following (X)HTML elements:
    • <frameset>    <frame>    <noframes>    <base>    <iframe>
  • Demonstrate the correct use and syntax of the following frame attributes:
    • Element: frameset
      Attributes: cols    rows
    • Element: frame
      Attributes: frameborder    id    marginheight    marginwidth    name    noresize    scrolling    src
    • Element: base
      Attributes: target
    • Element: iframe
      Attributes: align    frameborder    height    longdesc    marginheight    marginwidth    id    scrolling    src    width
