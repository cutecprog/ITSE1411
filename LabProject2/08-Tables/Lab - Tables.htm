Learning Outcomes1

  • Demonstrate the following (X)HTML programming skills:
    • Create a text table.
    • Create a graphical table.
    • Create table headers and captions.
    • Create table cells containing images.
    • Control the appearance of a table and table text.
    • Create table cells that span several rows or columns.
    • Create empty table cells.
    • Use nested tables to enhance page layout design.
  • Demonstrate the correct use and syntax of the following (X)HTML elements:
    • <table>    <tr>    <td>    <th>    <caption>    <thead>    <tbody>    <tfoot>    <colgroup>    <col>
  • Demonstrate the correct use of inline and embedded CSS styles.
  • Demonstrate the correct use and syntax of the following table related attributes & CSS Properties:
    • Element: <table>
      Attributes: summary    width    border    rules    frame    cellspacing    cellpadding
    • Element: <table>
      CSS Properties: object    display    float    background-color    background-image
    • Element: <caption>
      Attribute: align
    • Element: <tr>
      Attributes: align    valign
    • Element: <tr>
      CSS Properties: background-color    background-image
    • Elements: <td>    <th>
      Attributes: align    valign    rowspan    colspan
    • Elements: <td>    <th>
      CSS Properties: background-color    background-image    width    height
    • Elements: <colgroup>    <col>
      Attributes: align    span    width

Lab 1 (66 pts)

  1. Create a new Web page as shown in Figure 1.
    (2) Save the file in the 08-Tables folder with the filename: 01_tables.htm
  2. (18) General page requirements for the document:  01_tables.htm
    1. (10) Your document must pass validation:
      1. (5) W3C Markup Validation
      2. (5) W3C CSS Validation
    2. (2) Use the (X)HTML 1.1 DTD.
    3. (2) Insert two <meta> elements to identify the author (you) and document description.
    4. (2) Place your name in the page title followed by: Tables-1
      For example, Sally Student - Tables-1
    5. (2) Insert at least 2 comments in the document to describe the code.
  3. (5) Create a table (3 columns by 8 rows) to look like the document in Figure 1. Be sure to type your name in the indicated place.
  4. (19) (X)HTML Specifications:
    1. (2) Table Caption: Web Site Development Team - (your name here)
    2. (2) Caption displays: above the table.
    3. (2) First row: header row.
    4. (2) First column: header column.
    5. (2) Second row: header row.
    6. (3) First row: spans 3 columns.
    7. (3) 2nd row, 1st column spans 3 rows.
    8. (3) 5th row, 1st column spans 4 rows.
  5. (22) CSS Specifications:
    1. (5) Use embedded CSS styles to format the table.
    2. (3) Center align the table.
    3. (3) Table border is medium width, black color, and solid style.
    4. (2) Table text content: center aligned.
    5. (3) Cell borders are thin width, maroon color, and solid style.
    6. (2) First row: yellow background color.
    7. (2) First column: olive background color.
    8. (2) Add a teal background color to the second row.
