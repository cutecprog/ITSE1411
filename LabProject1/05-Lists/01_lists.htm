XHTML Lists (Term & Definitions)

List Item Element
List item element is a container element that encloses text that the browser treats as a list item in either an ordered or unordered list.
Ordered List Element
Ordered list ellemenet is a container element that encloses a block of text that the browser treats as a numbered list. Programmers use ordered lists to display ordered items or sequential steps.
Unordered List Element
Unordered list element is a container element that encloses a block of text that the browser treats as a bulleted list. Programmers use an unordered list to provide a logical organization for the displayed text.
Definition List Element
Definition list element is a container element that encloses a block of terms and their definitions. Programmers use definition lists to display dictionary-style information.
